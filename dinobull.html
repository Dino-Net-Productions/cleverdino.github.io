About DinoBull

DinoBull is a fork of Pitbull, which itself was a Hotline client+server+tracker application. DinoBull removes limitations such as being able to transfer files over 4GB in size, making it a powerful communication and file-sharing platform.

Key features of DinoBull include:

  • Ability to transfer files larger than 4GB
  • Discord-like appearance to differentiate it from Pitbull
  • Terminal-based server console
  • Enhanced encryption (8192-bit keys)
  • Customizable user interface with options to hide channel and user lists
  • Based on Pitbull Pro 4.6.2 fork source, with DinoBull-specific enhancements

Pitbull Legacy

DinoBull builds upon the rich history of Pitbull, which dates back to at least 2001. Pitbull went through numerous iterations and improvements before DinoBull forked from it. Some notable features inherited from Pitbull include:

  • Advanced file transfer capabilities
  • Robust chat and messaging systems
  • News and threaded discussions
  • User management and administration tools
  • Customizable interface and user experience

For a detailed history of Pitbull versions prior to the DinoBull fork, please refer to the Pitbull changelog.
